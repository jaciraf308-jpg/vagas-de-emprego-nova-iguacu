# Vagas de Emprego Nova Iguaçu

Este site é um portal simples de vagas que pode ser publicado em um serviço de hospedagem estática.

## Como publicar

### Opção 1: GitHub Pages
1. Crie um repositório no GitHub.
2. Envie os arquivos `index.html`, `styles.css` e `script.js` para o repositório.
3. No repositório, vá em `Settings` > `Pages`.
4. Selecione a branch principal (`main` ou `master`) e a pasta `/root`.
5. Salve e aguarde o link público aparecer.
6. O site ficará disponível em `https://<seu-usuario>.github.io/<nome-do-repositorio>/`.

### Opção 2: Netlify
1. Crie uma conta gratuita em https://www.netlify.com.
2. Conecte ao repositório GitHub ou faça upload manual dos arquivos.
3. Configure um site estático e publique.
4. Você pode definir um subdomínio como `vagas-de-emprego-nova-iguacu.netlify.app`.

### Opção 3: Vercel
1. Crie uma conta gratuita em https://vercel.com.
2. Importe o projeto do GitHub.
3. Use o deploy padrão para sites estáticos.
4. O Vercel terá um domínio público automático do tipo `nome-do-projeto.vercel.app`.

## Observação
- Para usar o link público com o nome `Vagas de Emprego Nova Iguaçu`, escolha um serviço que permita customizar o subdomínio ou use um domínio próprio.
- Se quiser, posso ajudar a montar o repositório Git local ou gerar o deploy no GitHub Pages passo a passo.
